# Employee Management System

A secure Employee Management System developed using Java Web Technologies following the MVC architecture.

---

## Features

- User Login
- Session-Based Authentication
- Authentication Filter
- Add Employee
- View Employees
- Search Employee
- Update Employee
- Delete Employee
- Logout

---

## Tech Stack

- Java
- JDBC
- Servlet
- JSP
- JSTL
- MySQL
- Apache Tomcat 8.5
- HTML
- CSS
- Eclipse IDE

---

## Project Architecture

Controller
↓
Service

DAO
